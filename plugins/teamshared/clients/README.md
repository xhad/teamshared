# Reference MCP snippets (manual setup)

These files are **documentation and copy-paste examples**. The curl installer
uses templated snippets under ``install/`` (served at ``/install/assets/*``).

| Harness | Installer template | Reference here |
|---|---|---|
| Cursor | ``install/cursor/mcp.json`` + ``rules/teamshared.mdc`` | ``protocol.md`` |
| Hermes | ``install/hermes/mcp.yaml`` | ``hermes.config.yaml`` (example URLs) |
| Claude | ``install/claude/mcp.json`` | ``claude-desktop.json`` (remote + local stdio) |
| Codex | ``install/codex/mcp.toml`` | — |
| Pi | ``install/pi/mcp.json`` | ``protocol.md`` |
| OpenClaw | generated by ``install.sh`` | ``../src/teamshared/clients/openclaw.md`` in repo |

**Memory rule:** ``../rules/teamshared.mdc`` (canonical). Plain-markdown twin:
``../rules/teamshared.md``.

**Agent protocol:** ``protocol.md`` — paste into Hermes SOUL, Claude project
instructions, etc. Same file is served as ``/install/assets/hermes/protocol.md``.
